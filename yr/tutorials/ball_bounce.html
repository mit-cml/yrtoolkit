Introduction to Environment

There are two main components to the App Inventor environment: the Designer and the Blocks Editor.

The Designer allows you to add components to your app and lay out what it will look like.

The Blocks Editor allows you to code your app to give it functionality--that is, make it do stuff!

You can toggle between the two screens using the buttons in the top right corner.

Designer Components

Within the Designer, there are five windows: Palette, Viewer, Components, Media, and Properties.

The Palette is where all available components are. You can drag and drop any component from the Palette into the Viewer.

The Viewer is where you can see what the app will look like and is where you drag components.

The Components window shows all components that have been added to the app. Note that the Screen component is automatically listed as a component.

The Media window shows any uploaded media files, like images and sounds.

Finally, the Properties window allows you to view or change any of the properties (or characteristics) of the currently selected component.

Set the Screen Properties

Before we add any components, uncheck the Scrollable property so the app does not extend outside the window of the mobile device.

Add a Canvas Component

From the Drawing and Animation drawer in the Designer, drag in a Canvas component. Then, set its Width and Height properties to "Fill Parent".

Add a Ball Sprite

Drag a Ball sprite from the Drawing and Animation drawer and drop it onto the Canvas. Then, set its Radius property to 10.

Ball1.Flung

In the Blocks Editor,click on Ball1 and click and drag when Ball1.Flung to the Viewer.

Click on Ball1 again. Click and drag the set Ball1.Heading to block and add it to when Ball1.Flung.

Also, drag a set Ball1.Speed to block to when Ball1.Flung.

Go to the when Ball1.Flung block. Hover over "heading" and click on get heading. Snap it to the set Ball1.Heading to block.

Do the same for get speed, snapping it to the set Ball1.Speed to block.

Test it out!

Try it out on your device, or in the emulator. If you're not connected yet, follow the Setup instructions above to connect and test your app.

You'll notice that it gets "stuck" on the edge of the screen. Next, we'll add an event to make the ball bounce off the edge of the screen.

Make the Ball Bounce

Click on Ball1 again and drag out the when Ball1.EdgeReached block. Then click on Ball1 again and connect the call Ball1.Bounce block to when Ball1.EdgeReached.

Go to the when Ball1.EdgeReached block. Hover over "edge" and click on get edge. Snap it to the call Ball1.Bounce block.


Test your app out again. The Ball should bounce off the edges now!.

Extend your app

Try to extend your app using these suggestions or come up with your own!

  • Change the color of the ball based on how fast it is moving or which edge it reaches
  • Scale the speed of the ball so that it slows down and stops after it gets flung
  • Give the ball obstacles or targets to hit
  • Introduce a paddle for intercepting the ball, like a Pong game
  • Check out the Mini Golf Tutorial, which uses the Ball1.Flung event too.

Keep exploring! See what you can come up with!

Now you've made a fun ball bounce game app!
